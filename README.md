# Guides

Статические guide-страницы для игровых прохождений. Каждый гайд — отдельная папка с `index.html`, все страницы используют единый шаблон, общие стили и скрипты.

**Stack:** HTML + CSS + Vanilla JS. Без фреймворков, без сборки, без сервера.

**Хостинг:** GitHub Pages из папки `docs/` на ветке `main`.

## Возможности

- Тёмная тема, mobile-first дизайн
- Якоря на шаги и оглавление
- Чекбоксы прогресса с сохранением в localStorage
- Возврат к последнему месту
- Режим «Только сюжет»
- Глобальное скрытие спойлеров
- Копирование ссылки на конкретный шаг
- Красивое превью ссылки (Open Graph)

## Структура

```
docs/
  assets/css/     — общие стили (reset, theme, guide)
  assets/js/      — общие скрипты (progress, ui, guide)
  assets/icons/   — favicon и иконки
  templates/      — шаблон для новых гайдов
  guides/         — папки гайдов (каждая с index.html)
  404.html        — страница ошибки
```
